## ydcli-rs

有道词典命令行版，一个简易的rust实现

相比无道词典，本项目没有用Server, Client 模式，内存占用更少，且输出更贴近于无道词典
完整模式。


![](https://raw.githubusercontent.com/charleschetty/ydcli-rs/main/shots/1.png)

### 用法： 

```sh 
ydcli-rs <word>
```
